About Us

Student Management System is software which is helpful for students as well as the school authorities. In the current system all the activities are done manually. Its time saving and scalable. Our Student Management System deals with the various activities related to the students In the software we can register as a user and user has two types student and administrator. Administrator has the power to add new user and can edit the students details entered. A admin can add students record ,attendance status with department wise. All students can search his/her basics details and attendance status with there respective roll numbers.

Objectives-

The main objective of the project is to design and develop a user friendly-system. the following objectives are-

  1. Easy to use and an efficient computerized system.
  2. To develop an accurate and flexible system, it will eliminate data redundancy.
  3. To study the functioning of Students management System.
  4. To make a software fast in processing, with good user interface.
  5. To make software with good user interface so that user can change it and it should be used for a long time without error and maintenance.
  6. To provide synchronized and centralized farmer and seller database.
  7. Computerization can be helpful as a means of saving time and money.
  8. To provide better Graphical User Interface (GUI).
  9. Less chances of information leakage.
  10. Provides Security to the data by using login and password method.
  11. To provide immediate storage and retrieval of data and information.
  12. Improving arrangements for students coordination.
  13. Reducing paperwork.
